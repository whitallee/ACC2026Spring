Preface

This chapter contains review questions and answers keyed to the chapter titled Java4030: Purpose of Framework Interfaces.

Once you study that chapter, you should be able to answer the review questions in this chapter.

The questions and the answers in this chapter are connected by hyperlinks to make it easy for you to navigate from the question to the answer and back again.

False. In general, there is no technical requirement for a specification of the behavior of the interface methods when implemented in a class. In fact, because a method that is declared in an interface is abstract, it specifically refrains from defining the behavior of the method. The interface definition simply declares the interfaces for all the methods that it declares.
