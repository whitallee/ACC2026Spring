The DrJava IDE, the IntelliJ IDEA Edu, BlueJ, and the Java Development Kit

In order to work with the material in this group of Programming Fundamentals chapters, you will need access to Oracle's Java Development Kit (JDK). Download and installation instructions are provided on that page.

You will also need access to a text editor, preferably one that is color-coded for Java programs. One such freely available text editor is named DrJava. DrJava is more than just a text editor. It is an Integrated Development Environment (IDE) that is designed for use by students learning how to program in the Java programming language. See A Quick Start Guide to DrJava for instructions on downloading and installing the DrJava IDE. The Quick Start Guide also provides instructions for using the DrJava IDE.

Another IDE that some students seem to like is the IntelliJ IDEA Edu (free as of 05/29/22). I haven't used this IDE so I can't vouch for it pro or con.

A third IDE that some students seem to like is BlueJ, described as "A free Java Development Environment designed for beginners, used by millions worldwide". Again, I haven't used this IDE so I can't vouch for it pro or con.

However, you don't need an IDE to write, compile, and run the programs in this course. All you need is a text editor that can be used to create files containing plain text and the JDK mentioned above. A Google search will identify several suitable text editors. For example, you might want to consider the use of Notepad++ as your text editor.
