MLO-3.3 Objectives, Read, and Do

Objective

Assess the student's ability to write programs that use lambda expressions.

Write a program that uses lambda expressions to perform the following operations on two incoming data values: addition, subtraction, multiplication, and division.

Write a program to determine whether or not a string is a palindrome.
