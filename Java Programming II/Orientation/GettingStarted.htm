Getting Started

The logical steps for getting started and completing this course are listed below.

1. Having read the Welcome page, you should continue by reading the syllabus in its entirety.

2. Next you should select Announcements in the navigation bar and read all of the announcements.

3. Next you should select Content in the navigation bar, select ORIENTATION, and complete your orientation.

4. Once you have completed orientation, select Content in the navigation bar and select Course Map to open the course map document in your browser. Study the relationships among the Course Level Objectives (CLOs), the Module Level Objectives (MLOs), and the Learning Activities(LAs). Then select Gradebook in the navigation bar. Each of the learning activities in the course map for which a grade will be assigned should appear in the Item column. The submission deadline for each learning activity is shown in the Due Date column. In the event that you don't understand these relationships, request clarification from your instructor.

5. Next you should select Content in the navigation bar, select Collections eBook, and then click the three-dot button. Download the zip file containing the free eBook. Extract the contents of the zip file into an empty folder. Open the file named index.htm, which contains a Table of Contents. The Table of Contents has links to each chapter in the eBook.

6. Go back to the Course Map and make certain that you understand how the reading assignments for chapters in the eBook correlate with the schedule for tests and assignments. Again, if  you don't understand something, ask your instructor for clarification.

7. Next you should select Content in the navigation bar and select LEARNING ACTIVITIES. This will open a page containing links to three CLO folders: CLO-1, CLO-2, and CLO-3.

8. Open the folder named CLO-1. There you will find four MLO folders. Each folder corresponds to a Module Learning Objective (MLO) in the course map.

9. Open the folder named MLO-1.1 and open the page named Objectives, Read, and Do. Study the contents of that page in order from top to bottom. Then close that page and complete Quiz01 and Asg01 in whichever order you prefer.

10. Having completed Quiz01 and Asg01, go back and select LEARNING ACTIVITIES, CLO-1, and MLO-1.2 and repeat the process to complete Quiz02 and Asg02.

11. Continue to repeat this process until you have completed Quiz04, and Asg12. At that point, you will have finished the course.

As you complete these steps, be mindful of the schedule shown on the Gradebook page to avoid missing a deadline. As further insurance against missing a deadline, you are encouraged to take your tests and submit your assignment early.

Have a good semester and be sure to ask questions if you don't understand something.

Prof. Baldwin
