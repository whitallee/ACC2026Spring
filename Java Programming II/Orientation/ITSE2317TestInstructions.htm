Instructions for Taking Online Tests

This course requires that you take one orientation test and four module quizzes. The due dates for all of the tests and quizzes can be found by selecting Gradebook on the navigation bar.

An orientation test

Test00 is an open-book, open-computer, online Blackboard (Bb) test. You may take the test as many times as you need from any computer that allows you to log into your Bb course.

The test contains one question. The purpose of the test is to confirm that you have completed your online orientation. Do the following to complete the test.

  1. Select Orientation on the main Content page.
  2. Scroll down and select Orientation Test00.
  3. Click the Start attempt Y button on the bottom right, where Y is the number of the attempt for the test.
  4. Click the Start Timer button on the popup window. This will expose a page showing a single test question.
  5. Read the question carefully and select either True or False.
  6. Click the Submit button on the bottom-right side of the page.
  7. Click the Submit button on the popup window.
  8. Click either the Close button or the Download button in the popup window.

The instructions listed above were valid as of 06/06/2024. The details may vary over time as the Bb software is updated.

Module quizzes

There is a take-home, open book, open computer, online Bb test associated with each of the first four Module Learning Objectives (see the Course Map on the main Content page.) The questions on each quiz are based on the eBook study material specified for the corresponding Module Learning Objective (MLO).

You can take the quiz an unlimited number of times until the due date has passed. To take a module quiz, do the following:

  1. Select LEARNING ACTIVITIES on the main Content page.
  2. Select the CLO of interest (CLO-1 for example).
  3. Select the MLO of interest (MLO-1.1 for example).
  4. Scroll down the page until you see a link to QuizXX where XX is the number of the quiz.
  5. Select the link to the quiz. This will expose information about the quiz on the right side of the page.
  6. Select Start Attempt Y where Y is the number of the attempt. This will expose the quiz questions on the left side of the page.
  7. Answer the quiz questions and click the Submit button on the lower-right side of the page.
  8. Click the Submit button on the popup window.
  9. Click either the Close button or the Download button in the popup window.

Your grade and some Assessment Results should appear on the right side of the page. You may need to scroll down to expose that information.

These instructions were valid as of 06/06/2024. The details may change over time as the Bb software is updated.

Alternatively you can skip the first few steps in the above list by selecting Shortcuts on the main Content page and then selecting the quiz of interest.
