Compiling and Running Java Programs from the Command Line

You don't need an IDE to compile and run Java programs. Once you get things set up, it is very easy to compile and run them from the command line. All you need is the Java Development Kit (jdk) that you can download from Oracle and a text editor. Detailed instructions for Windows are provided below. If you are using a different operating system, you will need to translate the instructions into that operating system.

Preparation

  1. Make certain that you can see file extensions such as .java, .bat, .txt, etc. If not, you may need to adjust your preferences.
  2. Download and install the Java SE Development Kit from Oracle.
  3. Create an empty folder somewhere on your disk
  4. Create an empty text file named aacmd.bat in that folder. (Note .bat and not .txt.)
  5. Use your text editor to put the following line of text in that file and close the file:
    cmd
  6. Double click that file. This will cause a command-line window to open with a command prompt shown by the > character.
  7. Type the following at the command prompt and press Enter:
    javac -version
    This should display text that shows the version of the Java compiler that you have installed.
  8. Type the following at the command prompt and press Enter:
    java -version
    This should display text that shows the version of the Java Virtual Machine (JVM) that you have installed
  9. If everything has worked to this point, this confirms that you have Java properly installed and ready for use. If not, you might want to seek help from the ACC CSIT tutors.

Compiling and running programs

  1. Create an empty text file in that same folder named HelloWorld.java. (Note .java and not .txt.) This is a simple Java source code file.
  2. Use your text editor to enter the following Java code in the file:
    class HelloWorld {
    public static void main(String[] args) {
    System.out.println("Hello, World!");
    }//end method
    }//end class
  3. Double click the file named aacmd.bat to open the command-line screen.
  4. Enter the following command at the command prompt:
    javac  -cp .; HelloWorld.java
    If there are no errors, the command prompt will reappear. If you examine the folder, you will see that it now contains a new file named HelloWorld.class. This is a compiled Java file ready to run.
  5. Enter the following command at the command prompt:
    java -cp .; HelloWorld
    The following output should appear on the screen:
    Hello, World!
  6. If everything has worked to this point, this confirms that you have successfully created, compiled, and executed a simple Java program. If not, you might want to seek help from the ACC CSIT tutors.
  7. From this point forward, each time you update your source code file or create a new source code file, you should execute the following commands at the command prompt:
    del *.class
    javac -cp .; fileName.java
    java -cp .; fileName
