Introduction

This document provides a map of:

  • Course Level Objectives (CLOs)
  • Module Level Objectives (MLOs)
  • Learning Activities (LAs)

The alignment among CLOs, MLOs, and LAs is shown by the hierarchical structure of the map.

The LAs that require reading refer to chapters in the Ebooks that you can download from Blackboard (Bb).

The required schedule for the learning activities that will be graded is provided in the Tools/My Grades page in your Bb course.

The purpose of each LA that requires reading is to learn the concepts covered in the MLO under which it appears.

The purpose of each quiz is to demonstrate knowledge of the concepts covered in the MLO under which it appears.

In general, the purpose of each assignment is to demonstrate a practical application of the concepts in the MLO under which it appears. The specific purpose of each assignment is explained in the introduction to the assignment when you select the Assignments link in the Bb course menu.

CLO-1 Demonstrate the ability to make effective use of object-oriented polymorphism and the Java Collections Framework.

MLO-1.1

Overview

Basics of the Java Collections Framework.

Objective

Write a computer program that demonstrates basic knowledge of the Java Collections Framework.

Identify:

  • TreeSet class
  • Type Object
  • Search speed differences between TreeSet and ArrayList objects
  • Duplicate values in TreeSet objects
  • Duplicate values in ArrayList objects
  • Relationship between the List interface and the Collection interface
  • Behavior of an Iterator object
  • Methods declared in the Iterator interface

Read and Do

Read chapters Java4010 through Java4040r in the Collections eBook.

Complete Quiz01

Complete Asg01

MLO-1.2

Overview

More advanced features of the Java Collections Framework.

Objective

Write a computer program that demonstrates knowledge of

  • core collection interfaces
  • duplicate elements
  • ordered collections
  • sorted collections
  • interface specialization

Identify:

  • The basic purpose of the core collection interfaces in the Java Collections Framework
  • An ordered collection
  • The result of populating TreeSet and ArrayList collections with the same values
  • The meanings of ordered and sorted
  • Duplicate elements in Set and List objects
  • The order of elements in a List object
  • The interface hierarchy for the Java Collections Framework
  • The Comparable interface
  • The Comparator interface
  • The compareTo method

Read and Do

Read chapters Java4050 through Java4080r in the Collections eBook

Complete Quiz02

Complete Asg02

Overview

The Comparator interface.

Objective

Write a computer program that demonstrates basic knowledge of the Comparator interface.

Identify:

  • The natural sorting order
  • The sorting order established by a Comparator
  • The Comparator as an alternative to the Comparable interface
  • The general behavior of the compare method of a Comparator object

Read and Do

Read chapters Java4090 through Java4110r  in the Collections eBook

Complete Quiz03

MLO-1.4

Overview

Sorting arrays and collections.

Objective

Write a computer program that demonstrates knowledge of sorting arrays and collections.

Identify:

  • The toArray method and a Vector object
  • The distinction between Collections and Collection
  • The natural ordering of the elements in a collection as imposed by the objects' own compare method
  • The use of a Comparator object to sort the contents of an array
  • The methods of the Collections class
  • The purpose of the class named Arrays
  • The LinkedList class

CLO-3 Demonstrate the ability to apply functional programming in Java.

MLO-3.1

Overview

Functional programming, streams, default methods, and method references.

Objective

Assess the student's ability to write code dealing with Java streams.

Write a computer program that demonstrates how to use streams to sort the String data in an incoming array object in descending order and return the sorted data in an object of type List.

Read and Do

Read the following lessons in the Functional Programming eBook:

  • Preface
  • Introduction to Functional Programming
  • Streams, Default Methods, and Method References

Complete Asg09

MLO-3.2

Overview

Functional interfaces.

Objective

Assess the student's ability to write programs that use functional interfaces.

Write a program that uses functional interfaces to perform the following operations on incoming data:

  • Determine the length of a string.
  • Test an integer for even or odd.
  • Concatenate two strings.

Read and Do

Read the Functional Interfaces lesson in the Functional Programming eBook.

Complete Asg10

MLO-3.3

Overview

Lambda expressions.

Objective

Assess the student's ability to write programs that use lambda expressions.

Write a program that uses lambda expressions to perform the following operations on two incoming data values: addition, subtraction, multiplication, and division.

Write a program to determine whether or not a string is a palindrome.

Read and Do

Read the Lambda Expressions lesson in the Functional Programming eBook.
